# {{projectName}}-frontend

A **BOT Chain** frontend scaffolded by `bot-cli`. This template uses **React 18**, **RainbowKit**, **wagmi v2**, **viem**, and **Tailwind CSS v4**.

## Prerequisites

- **Node.js** >= 18
- **pnpm**, **npm**, **yarn**, or **bun** (pnpm recommended)
- **Foundry** (optional — for Solidity contract compilation):
  ```bash
  curl -L https://foundry.paradigm.xyz | bash && foundryup
  ```

## Quick Start

```bash
# Install dependencies
{{installCommand}}

# Copy environment variables
cp .env.example .env
# Edit .env — at minimum set VITE_BOT_CHAIN_ID and VITE_BOT_RPC_URL

# Start dev server
{{devCommand}}
```

## Environment Variables

| Variable | Description | Default |
|---|---|---|
| `VITE_BOT_CHAIN_ID` | Target chain ID (677 = Mainnet, 968 = Testnet) | `968` |
| `VITE_BOT_RPC_URL` | RPC endpoint for the target network | `https://rpc.bohr.life` |
| `VITE_BOT_MAINNET_RPC_URL` | RPC for BOT Mainnet | `https://rpc.botchain.ai` |

## Features

### Wallet Connection (RainbowKit)
Click **Connect Wallet** in the navbar — RainbowKit opens a wallet selection popup with MetaMask, Rainbow, Coinbase Wallet, WalletConnect, and any other installed EIP-1193 provider.

### Contract Deploy & Interact (from UI)
Once a wallet is connected, the **Contract Playground** section appears. It reads compiled contract artifacts (injected by `botdev init` when Forge is installed) and lets you:

1. **Deploy** — Click "Deploy Contract" → signs a deployment transaction via your connected wallet
2. **Read** — Auto-fetches `view`/`pure` function return values and displays them
3. **Write** — Fill in parameters and click "Send" to call state-changing functions

If you haven't compiled contracts yet, run in the root project directory:

```bash
cd contracts
forge build
cd ../frontend
```

Then restart the dev server — the compiled artifact will be picked up.

> **Note**: The `contract.json` file is auto-generated by `botdev init` when Forge is detected. If you modify your Solidity contracts, re-run `forge build` to update the artifact.

### Dark / Light Mode
Toggle in the navbar. Defaults to your system preference.

### Network Switcher
Connected wallet shows the current chain name. Click it to toggle between BOT Mainnet and BOT Testnet.
