# UI Unification Plan

## Context
The user has a reference project at `/Users/eldevode/botchain/my-bot-app` whose frontend is a simple, centered wallet-connect card. They want this exact look and feel to become the standard UI for every frontend template generated by `botdev init`.

## Goal
Replace the current BOT-branded landing pages in:
- `templates/vite-vanilla`
- `templates/vite-vue`
- `templates/vite-react`

with the same minimal centered-card UI used in the reference project.

## Reference UI
The reference project is at `/Users/eldevode/botchain/my-bot` and uses **React + RainbowKit + Tailwind CSS v4** with a neobrutalist / BOT-branded design. Key elements:

- **Navbar** with BOT logo, "BOT Chain" title, network badge, dark/light toggle, and RainbowKit wallet button.
- **Hero** section with "Build on BOT Chain" headline.
- **Wallet detail card** when connected (address, network, chain ID, balance, explorer + disconnect buttons).
- **Feature cards** grid: Fast Finality, EVM Compatible, Builder Ready.
- **Network details** cards for Mainnet and Testnet with "Open Explorer" links.
- **Footer** with bot-cli attribution.
- **Light / dark theme** support with CSS custom properties and `data-theme` attribute.

## Approach
1. Use the reference React files as the canonical design.
2. Copy the reference implementation into `templates/vite-react` (RainbowKit, theme context, Tailwind v4 styles).
3. Port the same markup, icons, and styling to Vue (`App.vue`) using `@wagmi/vue` + a lightweight theme toggle (no RainbowKit).
4. Port the same design to Vanilla (`main.ts`) using `@wagmi/core` + a lightweight theme toggle (no RainbowKit).
5. For all three templates provide a Tailwind v4 variant (`style.tailwind.css`) and a plain-CSS variant (`style.css`) that look identical.
6. Keep `botChain.ts` with the `botNetworks` export.
7. Update dependencies so React template uses `@rainbow-me/rainbowkit`; Vue/Vanilla keep `injected` connector.

## Verification
- Run `pnpm lint` and `pnpm test`.
- Rebuild CLI and install globally.
- Run `botdev init` for each frontend template (Vanilla, Vue, React) and confirm the generated UI matches the reference project.
- Ensure production builds still pass for all three templates.
